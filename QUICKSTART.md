# 🚀 Guia Rápido - QA Academy

Seu guia de 5 minutos para começar com a QA Academy.

---

## ⚡ Início Rápido

### 1️⃣ Clone ou Baixe o Repositório

```bash
# Via Git
git clone <URL-do-repositorio>

# Ou baixe como ZIP
# Clique em "Code" → "Download ZIP"
```

### 2️⃣ Abra a Planilha Principal

```
📁 Pasta do Projeto
├── QA Academy Spreadsheet - OSF Academy.xlsx  ← ABRA AQUI
├── README.md
├── PROJECT.md
└── images/
```

### 3️⃣ Escolha seu Nível

| Nível | Para Quem | Tempo |
|-------|-----------|-------|
| 🟢 **Iniciante** | Novo em QA ou testes | 4-6 semanas |
| 🟡 **Intermediário** | Conhece testes básicos | 8-12 semanas |
| 🔴 **Avançado** | Quer automação | 12-16 semanas |
| ⚫ **Especialista** | Liderança em QA | Contínuo |

### 4️⃣ Comece a Aprender!

1. Abra a planilha
2. Leia o módulo do seu nível
3. Faça os exercícios
4. Avance para o próximo

## 🛠️ Ferramentas Essenciais (Configure Hoje)

### Nível Iniciante
```
✅ Navegador web (Chrome/Firefox)
✅ Excel/Google Sheets
✅ Bloco de notas
```

### Nível Intermediário
```
✅ Postman (API testing)
✅ Java/Python instalado
✅ IDE (VS Code/IntelliJ)
```

### Nível Avançado
```
✅ Selenium
✅ Cypress ou Playwright
✅ Git & GitHub
✅ Jenkins ou GitHub Actions
```
