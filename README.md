# 🎓 OSF Digital Academy - QA Academy

Bem-vindo ao repositório da **QA Academy** da **OSF Digital Academy**! Este projeto contém documentação e recursos para ensino e aprendizado em Quality Assurance (QA) e testes de software.

---

## 📋 Sobre o Projeto

A **QA Academy** é uma iniciativa educacional da OSF Digital Academy focada em:

- 📚 **Formação em QA**: Metodologias e práticas de Quality Assurance
- 🧪 **Testes de Software**: Estratégias de teste, automação e qualidade
- 🎯 **Desenvolvimento de Competências**: Trilhas de aprendizado estruturadas
- 🤝 **Comunidade**: Colaboração e compartilhamento de conhecimento

## 🚀 Como Começar

### Pré-requisitos

- Microsoft Excel ou Google Sheets para visualizar a planilha
- Conhecimento básico de testes de software (recomendado)

### Passos Iniciais

1. **Clone o repositório**:
   ```bash
   git clone <URL-do-repositorio>
   cd PROJETOS\ OSF\ DIGITAL\ ACADEMY
   ```

2. **Abra a planilha**:
   - Localize o arquivo `QA Academy Spreadsheet - OSF Academy.xlsx`
   - Abra com Microsoft Excel, Google Sheets ou LibreOffice Calc

3. **Explore o conteúdo**:
   - Comece pelos módulos iniciais
   - Siga a trilha de formação recomendada
   - Pratique com os exercícios propostos

## 🛠️ Ferramentas Recomendadas

![Ferramentas QA](./images/ferramentas-qa.svg)
*Principais tecnologias utilizadas em Quality Assurance*

Algumas ferramentas essenciais para QA:

| Ferramenta | Descrição | Tipo |
|-----------|-----------|------|
| **Selenium** | Automação de testes web | Framework |
| **JUnit / TestNG** | Testes unitários | Framework |
| **Postman** | Testes de API | Ferramenta |
| **Jira** | Gerenciamento de testes | Plataforma |
| **TestRail** | Gestão de casos de teste | Plataforma |
| **Cypress** | Automação web moderno | Framework |
