# ❓ Perguntas Frequentes (FAQ) - QA Academy

Respostas para dúvidas comuns sobre a QA Academy.

---

## 🎓 Sobre Aprendizado

### P: Para quem é a QA Academy?
**R**: Para qualquer pessoa interessada em Quality Assurance, desde iniciantes sem conhecimento técnico até profissionais buscando especialização. Temos trilhas para todos os níveis.

### P: Qual é a duração do curso?
**R**: Depende do seu nível:
- **Iniciante**: 4-6 semanas
- **Intermediário**: 8-12 semanas  
- **Avançado**: 12-16 semanas
- **Especialista**: Desenvolvimento contínuo

Mas você aprende no seu próprio ritmo!

### P: É necessário ter experiência em programação?
**R**: Não é obrigatório para começar no nível Iniciante. Porém, ajuda bastante para níveis mais avançados onde automação é necessária.

### P: Posso começar em um nível intermediário?
**R**: Recomendamos começar do Iniciante, mas se você tem experiência em testes, pode pular. Veja o conteúdo do Nível 1 para decidir.

### P: Há certificados?
**R**: Atualmente, reconhecimento por conclusão. Certificações oficiais estão em desenvolvimento para v1.2.0.

### P: Posso acessar offline?
**R**: Sim! Baixe a planilha Excel e acesse qualquer hora. As imagens SVG também são arquivos locais.

## 💻 Ferramentas e Tecnologias

### P: Preciso usar Windows, Mac ou Linux?
**R**: Qualquer um funciona! Quase todas as ferramentas QA modernas são cross-platform. Documentamos em todos quando aplicável.

### P: Preciso pagar por alguma ferramenta?
**R**: Não! Recomendamos ferramentas com versões gratuitas. Alternativas open-source também estão documentadas.

### P: Qual linguagem de programação devo aprender?
**R**: Sugerimos:
- **Java**: Para automação web (Selenium, JUnit)
- **JavaScript**: Para web moderno (Cypress)
- **Python**: Para scripts e testes rápidos
- **C#**: Para .NET applications

Comece com qualquer uma!

### P: Posso usar outras ferramentas que não estão documentadas?
**R**: Absolutamente! Os princípios são universais. Adapte os aprendizados para suas ferramentas preferidas.

## 🚀 Como Começar

### P: Por onde começar?
**R**: 
1. Leia o README.md
2. Baixe a planilha QA Academy
3. Comece com a Seção 1: Fundamentos
4. Pratique os exercícios

### P: Quanto tempo devo dedicar por dia?
**R**: Recomendado 1-2 horas por dia. Mas qualquer tempo ajuda! O importante é consistência.

### P: Posso pular seções?
**R**: Alguns podem pular, mas não recomendamos. Cada seção prepara para a próxima. Se tiver dúvidas, volte e revise.

### P: Preciso fazer os exercícios?
**R**: Sim! Prática é essencial em QA. Exercícios consolidam conhecimento.

## ❌ Problemas Comuns

### P: A planilha não abre
**R**: 
- Limpe o arquivo temporário `~$QA Academy...`
- Tente abrir com Google Sheets
- Verifique se Excel/Sheets está instalado
- Use LibreOffice Calc (grátis)

### P: Imagens não carregam
**R**: 
- Clone o repositório completo (não apenas files)
- Verifique se a pasta `images/` existe
- Use caminho relativo `./images/`

### P: Não entendo o conteúdo
**R**: 
- Revise tópico anterior
- Abra issue com dúvida específica
- Pesquise termos técnicos
- Pratique exemplos

### P: Tenho problema diferente
**R**: Abra uma issue em: https://github.com/osfdigital/qa-academy/issues
